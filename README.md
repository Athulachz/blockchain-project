This experiment demonstrates how blockchain ensures data integrity and how digital signatures ensure authentication and security.

First, in the basic blockchain implementation, we created a chain of blocks where each block contains data, a timestamp, and a hash. Each block is linked to the previous one using a hash generated by SHA-256. This ensures that if any data is changed, the hash also changes, breaking the chain and detecting tampering.

Next, in the blockchain with digital signature using RSA algorithm, we added security by signing transactions using a private key and verifying them using a public key. If any transaction data is modified after signing, the signature verification fails, proving data integrity and non-repudiation.

Finally, in the advanced version, we extended the system by:

Creating multiple users with their own RSA keys
Adding multiple transactions per block
Verifying each transaction individually
Introducing a simple mining process (Proof-of-Work)

This simulates a real-world blockchain system where:

Transactions are securely signed
Blocks are linked using hashes
Any tampering is easily detected
